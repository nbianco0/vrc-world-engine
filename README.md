# PC Hardware Sentiment Analysis

**Status:** Actively in development

An end-to-end NLP project that collects Reddit discussions related to GPUs, CPUs, and prebuilt PCs and performs aspect-based sentiment analysis to identify consumer trends and common pain points.

## Tech Stack
- Python
- PRAW (Reddit API)
- SQLite
- NLP (VADER, Transformers planned)
- Streamlit, Plotly
